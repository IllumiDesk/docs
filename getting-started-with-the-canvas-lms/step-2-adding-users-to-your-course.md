# Step 2: Adding users to your course

## Adding Instructors and Students to Your Course

Once you have [completed the IllumiDesk installation](https://docs.illumidesk.com/v2.0/docs/overview-illumidesk-lms-jupyter-notebooks) you need to add at least two users to the course with the following roles:

* An Instructor
* A Learner

You may add other users, such as teacher assistants after you have created the course. Refer to your LMS's documentation for specific instructions on how to add users to your LMS courses.

{% hint style="info" %}
#### Group Membership

Through the use of [LTI Advantage Names and Roles](https://www.imsglobal.org/spec/lti-nrps/v2p0) IllumIDesk automatically obtains user group membership information to determine what privileges users have within IllumiDesk.
{% endhint %}

{% hint style="success" %}
#### Student Accounts

Student \(also known as Learner\) accounts are created automatically. There is no need for users to enroll directly with the IllumiDesk service, these are created using the user id associated with their LMS instance.
{% endhint %}
